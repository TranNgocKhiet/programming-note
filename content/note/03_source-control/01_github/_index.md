## Important note

- When rebase a branch, that branch cannot be deleted until finish rebase or stop rebase using
```git rebase --abort```

- Step to init a new repo
1. ```git init```
2. ```git remote add origin <NEW_URL>```

- Remove remote
```git remote remove <ORIGIN>```
